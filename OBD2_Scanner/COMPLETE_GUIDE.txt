PHASE 1: BUILD YOUR APK ONLINE
-------------------------------
1. Open browser and go to: https://appgenerator.com/android-studio-online/

PHASE 2: HOST APK FOR FREE
--------------------------
CHOOSE ONE OPTION:

OPTION A: GOOGLE DRIVE (Easiest)
1. Go to: https://drive.google.com
2. Click "New"  "File upload"
3. Select your downloaded APK
4. Right-click the uploaded file  "Get link"
5. Change to "Anyone with the link can view"
6. Copy the link

OPTION B: NETLIFY DROP (Fastest)
1. Go to: https://app.netlify.com/drop
2. Drag & drop your APK file
3. Get instant URL (e.g., https://random-name.netlify.app/app-debug.apk)

OPTION C: GITHUB PAGES (Permanent)
1. Create account: https://github.com
2. Create new repository
3. Upload APK file
4. Go to Settings  Pages
5. Enable GitHub Pages
6. Your URL: https://[username].github.io/[repo]/app-debug.apk

PHASE 3: TEST WITH YOUR VW POLO
--------------------------------
1. ON YOUR ANDROID PHONE:
   - Open the APK link from browser
   - Download the APK file
   - Tap to install (enable "Unknown sources" if needed)

2. IN YOUR VW POLO 2013:
   - Plug NIOT OBD adapter into OBD2 port
   - Turn ignition ON (engine can be off)
   - Bluetooth pairing should happen automatically

3. USING THE APP:
   - Open "NIOT OBD Scanner" app
   - Tap "CONNECT TO NIOT OBD"
   - Tap "SCAN FOR P0036 CODE"
   - View the diagnostic results!

 TROUBLESHOOTING:
- If APK won't install: Enable "Unknown sources" in Android settings
- If OBD won't connect: Check Bluetooth is on, adapter is properly plugged in
- If no codes found: Try with engine running (not just ignition on)
